# Techvolution Minor Project

![Techvolution](https://img.shields.io/badge/Status-Live-success)

A clean and responsive registration form designed for the **Techvolution 2024 Event**. This project includes comprehensive client-side form validation built with HTML, CSS, and Vanilla JavaScript.

## 🚀 Live Demo
**[Click here to view the live project](https://shubbhamsingh.github.io/Techvolution_Minor_Project/)**

## ✨ Features
- Fully responsive registration form.
- Real-time client-side validation for emails, phone numbers, and required fields.
- Interactive UI using Bootstrap.

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript
- Bootstrap 5
